# 饭团工坊官网设计文档

## 概述

为饭团工坊（Fantuan Workshop）——星露谷物语模组制作工具——构建官网及后台管理系统。官网面向中文星露谷模组作者群体，提供产品展示、下载、文档、公告等功能。后台管理系统提供内容管理、下载管理、用户消息、公告发布及数据统计能力。

## 技术栈

| 层级 | 技术选型 | 版本要求 |
|------|---------|---------|
| 后端框架 | Laravel 11 | PHP 8.4 |
| 模板引擎 | Blade | — |
| 前端样式 | Tailwind CSS 3.x | — |
| 数据库 | MySQL | 5.7+ / 8.0 |
| 后台面板 | Filament 3.x | — |
| 服务器 | Nginx | — |
| 部署环境 | 宝塔面板 | — |

## 视觉风格

- **主题**：深色高级风
- **背景色**：`#0a0a0f`（深黑蓝）→ `#16162a`（深紫黑）渐变
- **主色**：`#f0c040`（琥珀金）— 按钮、高亮、品牌色
- **次色**：`#4ade80`（翠绿）— 点缀、成功状态
- **文字层级**：白色（标题）→ gray-300（正文）→ gray-500（辅助文字）
- **动效**：淡入上移、卡片 hover 发光边框、星空粒子背景
- **圆角**：rounded-lg (8px) / rounded-xl (12px) 为主
- **字体**：系统无衬线字体 + 中文优先

## 后台管理系统

### 技术方案
- 使用 **Filament 3.x** 管理面板包
- 通过 `php artisan make:filament-user` 创建管理员账号
- 基于角色的简单权限控制（管理员 / 编辑者）

### 管理模块

#### 1. 内容管理（CMS）
- 页面内容管理（首页、功能页、关于页等区块内容可编辑）
- 支持富文本编辑器 (TinyMCE / Trix)
- 支持图片上传与管理

#### 2. 下载管理
- 添加/编辑软件版本
- 上传安装包（或填写外部下载链接）
- 管理更新日志
- 版本发布/下架状态

#### 3. 用户消息
- 查看用户通过联系表单提交的消息
- 标记已读/未读
- 回复（可选邮件回复或站内标记）

#### 4. 公告管理
- 发布博客/公告文章
- 支持 Markdown 编辑
- 设置封面图、标签
- 定时发布

#### 5. 数据统计
- 下载量统计（按版本/时间）
- 页面访问统计
- 基础图表展示（日/周/月维度）

### 后台界面
- 深色主题（与官网风格统一）
- 侧边栏导航
- 顶部栏：搜索、通知、用户菜单
- 响应式布局

## 数据库设计

### 主要数据表

```sql
-- 管理员用户表（Laravel 内置 users 表扩展）
users: id, name, email, password, role, created_at

-- 页面内容表
pages: id, slug, title, content, meta, published, created_at, updated_at

-- 版本发布表
releases: id, version, title, description, changelog, download_url, file_size, 
          platform, published, created_at, updated_at

-- 用户消息表
messages: id, name, email, subject, content, read_at, created_at

-- 博客文章表
posts: id, title, slug, excerpt, content, cover_image, tags, 
       published_at, created_at, updated_at

-- 访问统计表
page_views: id, url, ip, user_agent, visited_at

-- 下载记录表
downloads: id, release_id, ip, user_agent, downloaded_at
```
