**日期：** 2026-06-17

**状态：** 待审核

## 1. 设计目标

将饭团工坊官网从"功能完整但视觉平淡"升级为"简洁、高端、有设计感"的网站，同时保留温暖的手作风格和星露谷基因。

**核心原则：**
- **简洁** — 减少信息密度，增加留白，让内容呼吸
- **高端** — 精致的配色、字体、动效，提升视觉品质
- **温暖** — 保留手作感，不追求冷冰冰的"科技感"
- **一致** — 所有页面统一设计规范

## 3. 字体和排版（F3）

### 字体栈

```css
--font-display: 'Noto Serif SC', 'Source Han Serif SC', Georgia, serif;
--font-body: 'Noto Sans SC', 'Source Han Sans SC', -apple-system, BlinkMacSystemFont, sans-serif;
```

### 字号系统

| 用途 | 字号 | 行高 | 字重 |
|------|------|------|------|
| Hero标题 | `clamp(48px, 8vw, 72px)` | 1.1 | 700 |
| Section标题 | `clamp(28px, 5vw, 40px)` | 1.2 | 600 |
| 卡片标题 | `18px` | 1.3 | 600 |
| 正文 | `16px` | 1.7 | 400 |
| 描述文字 | `14px` | 1.6 | 400 |
| 小字/标签 | `12px` | 1.5 | 500 |

### 字间距

```css
--tracking-display: -0.02em;  /* 标题略紧凑 */
--tracking-body: 0;           /* 正文正常 */
--tracking-label: 0.08em;     /* 标签略宽松 */
```

## 6. 动效规范（E1）

### 缓动曲线

```css
--ease-out: cubic-bezier(0.22, 1, 0.36, 1);
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
```

### 时长

```css
--duration-fast: 0.2s;
--duration-normal: 0.3s;
--duration-slow: 0.5s;
```

### 具体动效

| 元素 | 动效 |
|------|------|
| 滚动淡入 | `opacity: 0 → 1; transform: translateY(24px) → translateY(0)`，0.5s ease-out |
| 按钮悬停 | `transform: translateY(-2px); box-shadow: shadow-gold`，0.3s ease-out |
| 卡片悬停 | `border-color: rgba(212,160,74,0.3); transform: translateY(-4px)`，0.3s ease-out |
| 链接悬停 | `color: var(--gold)`，0.2s ease |
| 导航滚动 | `height: 64px → 56px; background: rgba(18,18,20,0.9); backdrop-filter: blur(12px)`，0.3s ease-out |
| Hero截图 | `transform: scale(1.02)`（悬停），0.5s ease-out |

## 10. 其他页面规范

### 10.1 工具页（features.php）

- **标题区：** 居中，大标题 + 简短描述
- **工具列表：** 垂直排列，每个工具一行
  - 左侧：大图标 + 编号
  - 右侧：标题 + 描述 + 特性标签 + 截图
- **底部CTA：** 居中，"下载工坊"按钮

### 10.2 下载页（download.php）

- **标题区：** 居中，大标题 + "免费 · 开源 · Windows 64位"
- **最新版本：** 大卡片，居中
  - 版本badge
  - 版本号（大字号）
  - 描述
  - 元信息（SMAPI版本、文件大小、许可证）
  - 下载按钮
- **历史版本：** 列表，每个版本一行
  - 版本号 + 平台
  - 日志按钮 + 下载链接

### 10.3 关于页（about.php）

- **标题区：** 居中，大标题
- **故事：** 卡片，居中，故事文字
- **技术栈：** 网格，4列
  - 每个技术：图标/名称 + 类别
- **贡献者：** 可选，GitHub贡献者网格

### 10.4 联系页（contact.php）

- **标题区：** 居中，大标题
- **表单：** 卡片，居中
  - 姓名 + 邮箱（并排）
  - 主题（单行）
  - 内容（多行textarea）
  - 发送按钮（全宽）
- **成功状态：** 卡片，居中，✓图标 + "已发送"

### 10.5 博客页（blog.php）

- **标题区：** 居中，大标题
- **文章列表：** 网格，2列
  - 每篇文章：日期 + 标题 + 简短描述 + "阅读更多"链接
- **分页：** 底部，上一页/下一页按钮

### 10.6 文档页（docs.php）

- **标题区：** 居中，大标题
- **文档列表：** 网格，2列
  - 每篇文档：图标 + 标题 + 简短描述
- **搜索：** 可选，搜索输入框

## 11. 响应式断点

```css
--breakpoint-sm: 640px;   /* 小手机 */
--breakpoint-md: 768px;   /* 大手机/平板 */
--breakpoint-lg: 1024px;  /* 平板/小桌面 */
--breakpoint-xl: 1280px;  /* 桌面 */
--breakpoint-2xl: 1536px; /* 大桌面 */
```

### 移动端适配

| 断点 | 调整 |
|------|------|
| `< 768px` | 导航变为汉堡菜单，Hero变为单列，工具卡片变为单列，画廊变为单列 |
| `< 640px` | 进一步压缩间距，字号适当缩小 |

## 13. 注意事项

1. **保持PHP逻辑不变** — 只修改HTML/CSS，不修改后端逻辑
2. **保留现有截图** — 使用 `assets/images/` 中已有的截图
3. **保留安全功能** — `sendSecurityHeaders()` 等安全相关代码不变
4. **渐进增强** — 新设计应在所有浏览器中正常工作，动效可降级
5. **性能** — 字体通过 CDN 加载，确保首屏加载时间可控
