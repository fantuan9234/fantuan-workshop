# Stardew Mod Studio — 开发计划

## 项目概述

面向小白用户的星露谷物语模组制作工具，无需编程即可替换NPC肖像/行走图、编辑事件、添加地图、修改物品，最终导出SMAPI兼容模组。

**技术栈**: Electron + React 18 + TypeScript + Tailwind CSS + react-router-dom v6  
**构建工具**: electron-vite  
**设计规范**: 三栏式暗色布局 (200px侧边栏 + 主内容区 + 300px右侧面板)

## 已完成模块

### NPC 模块
- NPC卡片列表 (45人, 6列网格, 含Wiki真实肖像)
- NPC详情页 (生日/住所/可婚/描述)
- 肖像编辑器 (10种表情 × 3个场景 = 30格上传)
- 行走图编辑器 (14种状态 × 12帧, 支持游戏目录导入)
- 新建NPC全屏弹窗 (表单创建自定义NPC)

### 事件模块
- 事件卡片列表 (含步骤类型标签、心数要求)
- 事件编辑器 (属性表单 + 8种步骤类型时间线)
- 步骤内联编辑 (对话/移动/选项/动画/BGM/奖励/特效/传送)
- 可视化坐标网格 (8×4地图坐标快速定位)

### 全局布局
- 三栏布局 (侧边栏导航 + 内容区 + 右侧信息面板)
- Electron IPC 游戏目录选择 + 文件读取
- 暗色主题 + 自定义滚动条

## 待开发模块

### 1. 物品模块 — 当前开发中

**页面结构** (遵循设计规范):
- **上半部分**: "我的创作" — 用户自定义物品卡片网格
- **下半部分**: "游戏参考素材" — 官方物品分类展示 (带搜索和筛选)

**物品编辑器**:
- 左侧: 属性表单 (名称/英文ID/分类/售价/品质/描述/可赠送/可烹饪)
- 右侧: 48×48物品图标预览 + 上传自定义PNG + 物品浮窗效果预览

**数据规模**: 星露谷有1000+物品, 参考库选取约100个代表性物品

### 2. 地图模块

**页面结构**:
- 上半部分: "我的创作" — 用户地图卡片 (缩略图+名称+尺寸)
- 下半部分: "游戏参考" — 官方地图缩略图预览 (农场/鹈鹕镇/沙滩等)

**地图编辑器**:
- 左侧图层管理 (地形/建筑/路径/装饰/NPC出生点)
- 中间画布 (拖拽缩放 + 网格吸附)
- 右侧素材库 (瓦片选择器: 地板/水/树木/建筑)
- 可放置图块、标记触发点/传送点

### 3. 任务模块

**页面结构**:
- 上半部分: "我的创作" — 任务卡片 (标题+NPC头像+奖励图标)
- 下半部分: "游戏参考" — 官方任务参考

**任务编辑器**:
- 基础属性: 任务名/类型(收集/击杀/送货/钓鱼/自定义)/触发NPC/前置条件
- 目标列表: 可添加多个目标 (收集X个Y物品 / 击杀X只Y怪物)
- 奖励配置: 金币/物品/好感度
- 任务描述多段文本编辑器

### 4. 导出模块

**页面结构**:
- 项目概览统计 (NPC数/事件数/地图数/物品数)
- 逐项勾选导出内容
- 一键生成SMAPI兼容zip包

**导出逻辑**:
- 生成 manifest.json + content.json
- 肖像 PNG → `assets/Portraits/`
- 行走图 PNG → `assets/Characters/`
- 事件 JSON → `Content/Data/Events/`
- 地图文件 → `.tmx` 格式
- 物品 JSON → `Content/Data/`
- 输出 `.zip` → 提示放入 Mods 文件夹

## 全局功能

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 项目保存/加载 (.stardew-mod) | P0 | JSON序列化所有编辑状态, 含文件浏览器打开 |
| NPC搜索/筛选 | P1 | 搜索框 + 筛选: 可婚/性别/生日月份 |
| 事件步骤预览播放 | P2 | 模拟对话气泡/角色移动动画的预览模式 |
| 撤销/重做 | P2 | 全局 Ctrl+Z / Ctrl+Y |
| 右键上下文菜单 | P3 | 步骤/元素右键: 删除/复制/移动 |

```
物品模块 → 项目保存/加载 → NPC搜索筛选 → 地图模块 → 任务模块 → 导出模块
```
